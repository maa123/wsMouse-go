## 使い方

上記リンクから利用するAndroid端末にアプリをインストール

[Release](https://github.com/maa123/wsMouse-go/releases)からTrackPadBridge.exeをダウンロード

TrackPadBridge.exeをダブルクリックで起動

アプリを起動して表示されたQRコードを読み込む
